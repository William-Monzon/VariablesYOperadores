\#Variables y Operadores



\##¿Para qué se utilizan los operadores lógicos en programación?



Los operadores lógicos se utilizan para evaluar condiciones que se le da a un programa. Con lo operadores lógicos puedes tomar decisiones para aceptar o no algunos datos en conjunto o por separado. Los operadores lógicos son tres: AND (\&\&) que devuelve 1 o true cuando todas las condiciones se cumplen y devuelve 0 o false cuando una condición falla. OR (||) devuelve 1 o true cuando al menos una condición se cumple y devuelve 0 o false cuando ninguna condición se cumple. Y NOT (!) devuelve lo contrario ósea 1 o true cuando no se cumple la condición y 0 o false cuando no se cumple.



\##¿Por qué es importante declarar correctamente el tipo de dato de una variable?



Es importante porque así el compilador entiende que un dato es un numero, un carácter, una palabra o un true/false, mientras que al no declarar las variables el programa dará error porque el compilador no entenderá que dato es la variable. Existen varios tipos de datos, como el int, duoble, float, Sting, boolean, char. Cada tipo tiene su distinta funcionalidad según las necesidades del programador, como el int que solo es para enteros o el String que es para cadena de caracteres o palabras. En fin es muy importante declararlas porque así nuestro programa funcionará al 100% y no nos consumirá mucha memoria.
